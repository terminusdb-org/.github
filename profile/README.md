[TerminusDB](https://terminusdb.com/) is an an open source document-oriented graph database **for building collaborative applications**. TerminusDB links JSON documents in a powerful knowledge graph through a simple [document API](https://terminusdb.com/docs/document-insertion/). Use TerminusDB to enable Git-like collaborative workflows in your applications and make use of graph query power using both [GraphQL](https://terminusdb.com/docs/graphql-basics/) and our Datalog query language [WOQL](https://terminusdb.com/docs/woql-basics/). 

[TerminusCMS](https://terminusdb.com/terminuscms/) is a hosted platform built on top of TerminusDB. It is a headless content and knowledge management system for complex environments where data, content, and document curation is interconnected and collaborative. [Sign up takes a minute and is free](https://dashboard.terminusdb.com/).
